The design of this website

I have had along held desire to make this website reflect the “true” nature of the web:

  • The site itelf looks like a web of links, and the home page should be essentially a visual representation of its contents, with links
  • Each page should look like it’s a node in a graph, and navigating from one node to another should leave the reader with a visual memory of the navigation from one part of the graph to another
